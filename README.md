>安装说明：  


因为使用了大量PHP8特性，源码只能在PHP 8.0+ 环境中运行  

1.将xm.sql上传到数据库  
2.修改/lib/mysql里面的数据库信息  
3.上传源码到服务器或虚拟主机 (PHP 8+)
